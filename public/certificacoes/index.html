🏅 Sobre a Certificação A AZ-900 é a porta de entrada para o ecossistema Azure. Arquitetura Azure: Entendimento de Regiões, Zonas de Disponibilidade e Grupos de Recursos. Serviços Principais: Máquinas Virtuais, Azure Functions, SQL Database e Azure Storage.
